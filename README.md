# Bridgepoint Connections — NM2TECH Concept Redesign

Premium concept redesign demo for **Bridgepoint Connections**, created by [NM2TECH](https://nm2tech.com/) to showcase modern web design and AI-powered visitor experiences.

> This is **not** the official Bridgepoint Connections website. Official site: [bridgepointconnections.org](https://www.bridgepointconnections.org/)

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide icons
- Local knowledge-base chatbot (LLM/RAG-ready API shape)
- AI Pathway Finder demo

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Import the project in Vercel
3. Deploy (no required env vars for the demo)

Optional env vars for a future LLM/email integration are listed in `.env.example`.

## Content policy

All company facts, contact details, testimonials, and program information were taken from Bridgepoint's public website. The demo does not invent certifications, phone numbers, or claims.
